# System Diagrams

This folder contains core diagrams for the smart seat reservation system, illustrating structure, interactions, and workflows.

| Diagram Title               | Brief Description                                                                 |
|-----------------------------|-----------------------------------------------------------------------------------|
| ERD Diagram                 | Shows database entity relationships (e.g., `users`, `bookings` tables).|
| UML Class Diagram           | Defines core classes, attributes, methods, and their relationships.               |
| System Architecture Diagram | Outlines frontend, backend, database, and external service layers.                |
| Use Case Diagram            | Details interactions for User (student/teacher), Admin, and Lecturer/Tutor.       |
| Sequence Diagram            | Illustrates step-by-step flow of the seat reservation process.          |
